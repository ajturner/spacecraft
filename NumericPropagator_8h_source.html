O_SESSAME::Propagator
Encapsulation of the propagation of the spacecraft dynamics (attitude, orbital, etc.)The propagator is used to tie together the various spacecraft dynamics that are desired to be propagated. The user has the ability to set the time derivative equations, integrators, parameters, etc. as necessary. Furthermore, the Propagator class can be derived for more functionality.
