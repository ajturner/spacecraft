The user has the ability to set the time derivative equations, integrators, parameters, etc. as necessary. Furthermore, the Propagator class can be derived for more functionality.

O_SESSAME::Environment
Encapsulation of a spacecraft environment (disturbance forces, torques, central body, etc.)The Environment class is used to contain all the different aspects of the external environment of the spacecraft. This currently includes disturbance forces (gravity, drag, solar radiation pressure), disturbance torques (gravity gradient, magnetic dipole, fuel slosh), and central body (for constants)..
