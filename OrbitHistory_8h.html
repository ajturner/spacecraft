class  O_SESSAME::OrbitHistory
 Class for storing a time history of orbital states.The OrbitHistory class provides a container of previous OrbitState objects. The user may append new states, reset the collection of states, or return the orbit history in a matrix. If the user appends orbit states at time which earlier than the end of the current history, the current history is deleted from that point on, and the new orbit states are appended. More...
